ðŸ”¹ 2 Struktur Database
* Assets Table
Menyimpan semua aset/akun keuangan (Bank, E-wallet, Crypto, dll)
- assetId = Int (Primary Key, Auto Generate)
- assetName = String (Nama aset: "Bank BCA", "Bitcoin Wallet")
- assetType = String (Enum: BANK, E_WALLET, CRYPTO, GOLD, CASH)
- currentBalance = Double (Saldo saat ini dalam unit asli)
- balanceUnit = String (Unit: "IDR", "USD", "BTC", "gram")
- currencySymbol = String (Symbol: "Rp", "$", "â‚¿", "gr")
- accountNumber = String? (Nomor rekening - nullable)
- bankName = String? (Nama bank - nullable)
- lastPriceUpdate = Double? (Harga terbaru per unit dalam IDR - nullable)
- priceSource = String? (Sumber harga: "CoinGecko", "Manual" - nullable)
- isActive = Boolean (Status aktif/nonaktif - default true)
- sortOrder = Int (Urutan tampil di UI - default 0)
- createdDate = Long (Timestamp kapan dibuat)
- lastModified = Long (Timestamp terakhir diubah)

* Transactions Table*
Mencatat semua transaksi: income, expense, transfer
- transactionId = Int (Primary Key, Auto Generate)
- fromAssetId = Int (Foreign Key ke Assets - aset sumber)
- toAssetId = Int? (Foreign Key ke Assets - aset tujuan untuk transfer - nullable)
- transactionType = String (Enum: INCOME, EXPENSE, TRANSFER_IN, TRANSFER_OUT)
- transactionCategory = String (Kategori: "Food", "Transport", "Salary")
- transactionAmount = Double (Jumlah dalam unit aset sumber)
- transactionCurrency = String (Currency: "IDR", "USD", "BTC")
- convertedAmountIDR = Double (Nilai dalam IDR untuk laporan)
- exchangeRate = Double (Rate konversi saat transaksi - default 1.0)
- transactionTitle = String (Judul: "Makan siang", "Gaji bulanan")
- transactionNote = String? (Catatan tambahan - nullable)
- transactionLocation = String? (Lokasi transaksi - nullable)
- receiptImagePath = String? (Path foto struk - nullable)
- transactionDate = Long (Timestamp tanggal transaksi)
- createdDate = Long (Timestamp kapan data dibuat)

* Categories Table
Kategori transaksi (bawaan + custom user)
- categoryId = Int (Primary Key, Auto Generate)
- categoryName = String (Nama kategori: "Food & Drinks", "Transportation")
- categoryDescription = String? (Deskripsi kategori - nullable)
- isIncomeCategory = Boolean (true = income, false = expense - default false)
- categoryColor = Int (Warna untuk UI dalam format ARGB)
- categoryIcon = String (Nama icon atau emoji)
- isSystemCategory = Boolean (true = bawaan, false = custom user - default false)
- isActive = Boolean (Status aktif - default true)
- usageCount = Int (Berapa kali dipakai untuk sorting - default 0)
- createdDate = Long (Timestamp kapan dibuat)

*Budgets Table
Budget planning per kategori per bulan
- budgetId = Int (Primary Key, Auto Generate)
- budgetCategoryId = Int (Foreign Key ke Categories)
- budgetName = String (Nama budget: "Food Budget March 2024")
- budgetLimit = Double (Batas pengeluaran dalam IDR)
- budgetPeriod = String (Enum: MONTHLY, WEEKLY, DAILY)
- budgetMonth = Int (Bulan: 1-12)
- budgetYear = Int (Tahun: 2024)
- currentSpent = Double (Total pengeluaran saat ini - default 0.0)
- isActive = Boolean (Status aktif - default true)
- alertThreshold = Double (Persentase untuk notifikasi: 0.8 = 80% - default 0.8)
- createdDate = Long (Timestamp kapan dibuat)
- lastUpdated = Long (Timestamp terakhir diupdate)

* Price History Table
Menyimpan riwayat harga untuk crypto/gold/forex

- priceId = Int (Primary Key, Auto Generate)
- currencyCode = String (Kode mata uang: "BTC", "ETH", "XAU")
- priceInIDR = Double (Harga dalam IDR)
- priceInUSD = Double (Harga dalam USD)
- priceSource = String (Sumber: "CoinGecko", "GoldAPI", "Manual")
- priceDate = Long (Timestamp kapan harga diambil)
- isLatest = Boolean (Apakah ini harga terbaru - default false)

*Transfer Pairs Table
Menyimpan pasangan transfer untuk history dan quick transfer
- pairId = Int (Primary Key, Auto Generate)
- fromAssetId = Int (Foreign Key ke Assets - aset sumber)
- toAssetId = Int (Foreign Key ke Assets - aset tujuan)
- pairName = String (Nama pasangan: "PayPal â†’ Bank BCA")
- lastUsed = Long (Timestamp terakhir digunakan)
- usageCount = Int (Berapa kali dipakai - default 0)
- isQuickTransfer = Boolean (Tampil di quick transfer - default false)
- createdDate = Long (Timestamp kapan dibuat)

* Backup Metadata Table
Metadata untuk backup/restore
- backupId = Int (Primary Key, Auto Generate)
- backupName = String (Nama backup: "Backup_2024_03_15")
- backupPath = String (Path file backup)
- backupType = String (Enum: JSON, CSV, GOOGLE_DRIVE)
- backupSize = Long (Ukuran file dalam bytes)
- totalAssets = Int (Jumlah aset yang dibackup)
- totalTransactions = Int (Jumlah transaksi yang dibackup)
- backupDate = Long (Timestamp kapan backup dibuat)
- isAutoBackup = Boolean (Apakah backup otomatis - default false)

*Database Relations

*Assets â†’ Transactions
- One-to-Many: Satu aset bisa punya banyak transaksi
- Foreign Key: transactions.fromAssetId â†’ assets.assetId

*Categories â†’ Transactions
- One-to-Many: Satu kategori bisa dipakai di banyak transaksi
- Relasi via String matching: transactions.transactionCategory = categories.categoryName

*Categories â†’ Budgets
- One-to-Many: Satu kategori bisa punya beberapa budget (bulanan)
- Foreign Key: budgets.budgetCategoryId â†’ categories.categoryId

*Assets â†’ Transfer Pairs
- Many-to-Many: Aset bisa jadi sumber/tujuan di banyak pasangan transfer
- Foreign Key: transfer_pairs.fromAssetId & transfer_pairs.toAssetId â†’ assets.assetId

Potential Challenges & Solutions
- Challenge: Currency conversion accuracy

- Challenge: Large transaction lists
- Solution: Implement pagination dengan Room

- Challenge: Background price updates
- Solution: Use WorkManager dengan retry policy

- Challenge: Chart performance
- Solution: Limit data points, use efficient data structures
